# Patch Review Checklist

Before review:

```text
[ ] On main
[ ] Worktree clean
[ ] Package ZIP is flat-rooted
[ ] changed-files.txt matches expected paths
[ ] No dangerous shell patterns
```

Report:

```text
[ ] git status shown
[ ] changed files shown
[ ] untracked/new files shown
[ ] diff stat includes new files
[ ] full diff includes new files
[ ] only expected paths changed
```
